# ЕРПВ — шуточная страница электронной повестки

Розыгрыш для друзей: реалистичная страница «электронной повестки» в стиле государственного портала.

## Быстрый старт

```bash
npm install
npm run dev
```

Откройте [http://localhost:5173/generator](http://localhost:5173/generator), заполните форму, скопируйте ссылку и отправьте другу.

## Как пользоваться

1. Перейдите на `/generator`
2. Введите ФИО друга, дату рождения, дату/время явки и адрес военкомата
3. Нажмите «Сгенерировать ссылку»
4. Скопируйте и отправьте ссылку

Друг увидит страницу с загрузкой «проверки реестра», уведомлением об ЭП, документом повестки (PDF/печать), кнопкой «Войти через Госуслуги».

## URL-параметры

Можно собрать ссылку вручную:

```
/?name=Иванов+Иван+Иванович&birth=1998-05-15&date=2026-06-25&time=09:00&office=ВК+г.+Москвы
```

| Параметр | Описание |
|----------|----------|
| `name` | ФИО |
| `birth` | Дата рождения (YYYY-MM-DD) |
| `date` | Дата явки |
| `time` | Время явки |
| `office` | Адрес военкомата |

## Сборка

```bash
npm run build
npm run preview
```

Статика подходит для Vercel, Netlify, GitHub Pages.

## Деплой на GitHub Pages

Сайт: **https://f-l-e-m.github.io/gg/**

1. Запушьте код в ветку `main`
2. GitHub → репозиторий **gg** → **Settings** → **Pages** → Source: **GitHub Actions**
3. После успешного workflow в **Actions** сайт будет доступен по ссылке выше

| Страница | URL |
|----------|-----|
| Генератор | https://f-l-e-m.github.io/gg/generator |
| Розыгрыш | https://f-l-e-m.github.io/gg/?name=... |

Локально `npm run dev` работает как обычно на `http://localhost:5173/`. При сборке (`npm run build`) пути автоматически подстраиваются под `/gg/`.

## Важно

- Сайт **не собирает** реальные данные и **не** подключается к Госуслугам
- Используйте **только среди друзей** в шуточных целях
- Не используйте для мошенничества или введения людей в заблуждение незнакомцев

# ЕРПВ — шуточная страница электронной повестки

Розыгрыш для друзей: реалистичная страница «электронной повестки» в стиле государственного портала.

## Быстрый старт

```bash
npm install
npm run dev
```

Откройте [http://localhost:5173/generator](http://localhost:5173/generator), заполните форму, скопируйте ссылку и отправьте другу.

## Как пользоваться

1. Перейдите на `/generator`
2. Укажите дату/время явки и адрес военкомата
3. Нажмите «Сгенерировать ссылку»
4. Скопируйте и отправьте ссылку

Друг увидит страницу с загрузкой «проверки реестра», уведомлением об ЭП, документом повестки (PDF/печать), кнопкой «Войти через Госуслуги».

## URL-параметры

Можно собрать ссылку вручную:

```
/?date=2026-06-25&time=09:00&office=ВК+г.+Москвы
```

| Параметр | Описание |
|----------|----------|
| `date` | Дата явки |
| `time` | Время явки |
| `office` | Адрес военкомата |

## Сборка

```bash
npm run build
npm run preview
```

Статика подходит для Vercel, Netlify, GitHub Pages.

## Деплой на GitHub Pages (папка `/docs`)

Сайт: **https://f-l-e-m.github.io/registry/**

### Настройки GitHub

**Settings** → **Pages** → Source: **Deploy from a branch** → ветка `main` → папка **`/docs`**

Не используйте `/root` — там исходники, а не собранный сайт.

### Перед каждым пушем

```bash
npm run build:pages
git add docs
git commit -m "Update site"
git push origin main
```

Команда `build:pages` собирает проект, копирует результат в `docs/` и добавляет `.nojekyll` (чтобы GitHub не запускал Jekyll).

| Страница | URL |
|----------|-----|
| Генератор | https://f-l-e-m.github.io/registry/generator |
| Розыгрыш | https://f-l-e-m.github.io/registry/?date=... |

Локально `npm run dev` — `http://localhost:5173/`.

## Важно

- Сайт **не собирает** реальные данные и **не** подключается к Госуслугам
- Используйте **только среди друзей** в шуточных целях
- Не используйте для мошенничества или введения людей в заблуждение незнакомцев
